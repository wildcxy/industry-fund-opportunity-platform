# 前端自测清单

## 1. 构建与类型

1. 执行 `npm.cmd run build` 成功通过。
2. 不出现新的类型错误、路由错误或构建时页面崩溃。

## 2. 主链路检查

1. 首页可进入行业详情页。
2. 行业详情页点击“查看相关基金”后，基金发现页保留主题筛选。
3. 基金发现页切换筛选、排序、视图模式后，URL 查询参数同步变化。
4. 基金发现页可将基金加入对比与观察。
5. 基金对比页可展示 2 至 4 只基金的收益、风险、持仓和结论提示。
6. 我的观察页可稳定显示已添加的行业与基金，并支持移除与跳转。

## 3. PRD 对齐检查

1. 五大页面均具备页面目标对应的核心模块。
2. 所有评分、标签、排序和风险提示都带有说明位或口径描述。
3. 页面文案不出现“稳赚”“抄底”“确定性收益”等违规表述。
4. 首页、行业详情页、基金发现页、基金对比页、我的观察页均提供空状态或引导信息。

## 4. 响应式检查

1. 首页卡片与热力图在窄屏下不出现不可读重叠。
2. 基金发现页表格在移动端可横向滚动，卡片视图可正常换行。
3. 行业详情页按钮、结论卡和图表区在移动端不溢出。
