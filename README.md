# 行业基金机会捕捉平台

这是一个基于 `Next.js + Tailwind CSS + TypeScript` 的网页前端演示项目，用于展示“行业机会发现 -> 基金筛选 -> 基金对比 -> 加入观察”的完整产品链路。

## 当前包含

- 首页：市场概览、行业机会榜、重点行业卡片、行业热力图、热门基金映射
- 行业详情页：机会结论卡、趋势与资金图表区、催化事件、基金映射、口径说明
- 基金发现页：筛选、排序、表格/卡片双视图、URL 状态恢复
- 基金对比页：收益风险摘要、核心指标对比表、持仓与风格概览、对比结论提示
- 我的观察页：行业/基金观察分区、最近变化摘要、待跟踪机会提示
- Mock API：`/api/industries`、`/api/industries/[id]`、`/api/funds`

## 启动方式

1. 安装依赖：`npm.cmd install`
2. 启动开发环境：`npm.cmd run dev`
3. 浏览器访问：`http://localhost:3000`

## 目录概览

- `app/`：页面路由与 API Route
- `components/`：通用 UI 与交互组件
- `features/`：功能域交互组件
- `lib/`：格式化、存储、适配层
- `mock/`：样例数据
- `types/`：核心数据契约与页面模型
- `docs/`：PRD、架构说明、自测清单与产品决策文档
